# Task 2: Customer Segmentation Using K-Means Clustering

## Objective

The objective of this project is to segment customers based on their annual income and spending behavior using K-Means Clustering. These customer segments help businesses design targeted marketing strategies.

## Dataset

- Dataset: Mall Customers Dataset
- Source: Kaggle

## Approach

- Loaded and explored the dataset
- Performed data cleaning
- Conducted Exploratory Data Analysis (EDA)
- Selected relevant features
- Applied Feature Scaling
- Used the Elbow Method to determine the optimal number of clusters
- Built a K-Means Clustering model
- Visualized clusters using PCA
- Suggested marketing strategies for each customer segment

## Results

- Optimal Number of Clusters: **5**
- Successfully grouped customers into five meaningful segments.
- PCA visualization clearly separated customer clusters.

## Conclusion

K-Means Clustering effectively identified customer groups with different purchasing behaviors. These insights can help businesses create personalized marketing campaigns, improve customer engagement, and increase sales.
